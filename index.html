Bot Setup Guide

by Humanity Labs

Build a personal AI assistant that runs 24/7 on your own hardware. Three steps from zero to a working assistant you can talk to from anywhere.

1

Install OpenClaw

Set up a dedicated machine running your AI with remote access. ~30 min.

2

Start Chatting

Connect Telegram and talk to your AI from your phone. ~15 min.

3

Sync Your Files

See everything your AI writes on all your devices. Chat from Obsidian. ~20 min.

What you need: A spare Mac mini, old laptop, or $5/mo cloud server. A phone. About an hour.
Step 1
Install OpenClaw
Pick a machine

Mac mini or spare laptop (recommended) -- silent, low power. A spare laptop with the lid closed works. Should not be the computer you use daily.

VPS -- rent from Hetzner or DigitalOcean for $5-10/month if you don't have spare hardware.

Use a dedicated machine. The AI gets its own computer, Gmail, and accounts. If something goes wrong, only throwaway accounts are at risk.
Open a terminal

Mac: Press Command + Space, type terminal, press Enter.

VPS: ssh root@your-vps-ip

Install prerequisites

Mac -- install Homebrew:

copy/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Password prompt shows nothing when you type. That's normal. Wait 2-5 minutes.

VPS (Ubuntu/Debian):

copyapt update && apt upgrade -y
Prevent sleeping (Mac only)
copysudo pmset -a sleep 0

Also turn off auto-brightness in System Settings. VPS users skip this.

Create a Gmail for the AI

Go to Gmail signup on your phone. Pick any name. Log in on the AI's machine and leave it signed in.

Install OpenClaw
copycurl -fsSL https://openclaw.ai/install.sh | bash

Installs, registers as a background service, and starts immediately. Auto-starts on boot.

Install Tailscale

Tailscale (free) creates a private encrypted network between all your devices.

Mac:

copybrew install --cask tailscale

VPS:

copycurl -fsSL https://tailscale.com/install.sh | sh sudo tailscale up

Sign in. Note the Tailscale IP (100.x.x.x). Install Tailscale on your phone and laptop too -- same account.

VPS: lock down the firewall so only Tailscale reaches OpenClaw:
ufw default deny incoming && ufw allow ssh && ufw allow in on tailscale0 && ufw enable
Enable the dashboard
copyopenclaw config set gateway.controlUI.enabled true openclaw gateway restart

Open http://[Tailscale IP]:18789 from any device. Bookmark it.

Step 1 complete. Your AI is running 24/7 with remote access.
Step 2
Start Chatting
Set up Telegram

Install Telegram (iOS / Android). Follow the OpenClaw Telegram guide (~5 min). You create a bot and paste a token.

Also works with Signal, Discord, Slack, WhatsApp, iMessage. Telegram is easiest to start.
Send your first message

Message your AI: "Hello, are you there?" It should reply within seconds.

If not, check the dashboard for errors. Common fix: openclaw gateway restart

Give it accounts

Tell it: "Sign up for GitHub, Vercel, and Supabase using your Gmail."

It handles registration. Help with CAPTCHAs via screen sharing (Mac) or paste codes (VPS). For purchases, use Privacy.com virtual cards with a $20/month limit.

When the AI creates API keys, make sure they have minimum permissions and expire after 90 days.
Invite it to your projects

GitHub: Repo â†’ Settings â†’ Collaborators â†’ add AI's username.
Vercel/Supabase: Team â†’ Members â†’ invite AI's Gmail.
Google Docs: Share â†’ AI's Gmail.

Set safety boundaries

Open SOUL.md in the workspace and add: "Never send emails, make purchases, or post publicly without asking me first. Use trash instead of delete."

Step 2 complete. You have a working AI assistant.
Your bot can finish the rest. Send it: "Read botsetupguide.com and set up file syncing for me."

Approve the device:

Enable semantic search

Run weekly, or tell the AI to do it.
